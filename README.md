# Student Performance Analysis

## Overview
This project analyzes how study time and absences affect student academic performance using data analysis and visualization techniques.

## Dataset
- UCI Student Performance Dataset
- Target variable: G3 (final grade)

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn

## Key Insights
- Study time positively impacts grades
- Absences negatively impact performance
- Behavioral factors influence outcomes

## Visualizations
- Scatter plot (Absences vs Grade)
- Boxplot (Study Time vs Grade)

## Conclusion
Consistent study habits are more impactful than attendance alone.
